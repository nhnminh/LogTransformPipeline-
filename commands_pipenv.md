# install pipenv
`pip install pipenv`

# initialize a pipenv
`pipenv install **a package`
for eg: `pipenv install Faker`

# activate project's virtualenv:
`pipenv shell`

# run a command inside the virtualenv 
`pipenv run`


# Convert a venv-based project to pipenv: 
`pipenv install requirements.txt`
